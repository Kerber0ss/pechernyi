# 🪨 Pechernyi — Довідка

## Режими
| Режим | Що робить |
|-------|-----------|
| **lite** | Без вставних слів та хеджування. Повні речення, ввічливий тон |
| **full** | Скидати вставні слова, фрагменти, підмет опускати. Класичний печерний |
| **ultra** | Максимальне стиснення. Абревіатури, стрілки (X → Y), одне слово коли достатньо |

## Команди
- `/pechernyi` — увімкнути (full за замовчуванням)
- `/pechernyi lite|full|ultra` — обрати рівень
- `/pechernyi-commit` — стислий commit
- `/pechernyi-review` — одно-рядковий code review
- `/pechernyi:compress <filepath>` — стиснути markdown файл

## Активація
- `/pechernyi`, "печерний режим", "говори коротко", "менше токенів"
- "активуй печерний", "говори як печерний"

## Деактивація
- "стоп печерний", "звичайний режим"

## Скіли
- **pechernyi-commit** — Conventional Commits з українським описом
- **pechernyi-review** — Л<рядок>: 🔴/🟡/🔵/❓ коментарі
- **pechernyi-compress** — стискає markdown файли на ~46%
